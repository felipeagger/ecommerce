# ecommerce
An simplified e-commerce using top technologies to support high availability.

# Technologies

Made with microservices in **Quarkus (Java) and Golang** using:

* ElasticSearch
* DynamoDB
* PostgresSQL
* Apache Kafka

* Nginx (local APIGateway)
